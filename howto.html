Minecraft Resource Pack Generator

This tool allows you to create Minecraft resource packs easily online. You can customize textures and download them instantly.

How to use

  1. Open the editor
  2. Add blocks and textures
  3. Click generate
  4. Download the zip file

リソースパックの作り方

このツールでは簡単にMinecraftのリソースパックを作成できます。 ブロックを追加して、生成ボタンを押すだけでZIPとしてダウンロードできます。

Features

  • Free online tool
  • No installation required
  • Instant download
